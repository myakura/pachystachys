Resource Timing

読み込むリソースの時間を取得。 

// リソースのエントリ取得
performance.getEntriesByType('resource')
// エントリのイニシエータ
entry.initiatorType // 'img', 'css', 'script', etc.
// リクエスト開始までにかかった時間
entry.requestStart - entry.fetchStart

User Timing

任意に「マーク」して計測。 

// マークする
performance.mark('mark_start')
// マークしたエントリ取得
performance.getEntriesByType('mark')
// 計測を記録
performance.measure('mark_start', 'mark_end')
